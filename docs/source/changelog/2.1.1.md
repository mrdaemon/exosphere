# 2.1.1 - Hotfix

*Released November 01, 2025*

## Hotfix Release

Minimum version of Textual as a dependency was incorrectly set, in some
installations this can result in the Inventory screen crashing on access with
`Footer.__init__() got an unexpected keyword argument 'compact'`

Bumps Textual dependency minimum to v6.2.0 to resolve the issue.
